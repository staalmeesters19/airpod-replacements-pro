## 5. Migratiestrategie: airpodshandel.nl → repairpods.nl

#### Redirect Regels

```

airpodshandel.nl/product-categorie/pro-2-2/ → repairpods.nl/losse-airpods/airpods-pro-3/
airpodshandel.nl/product-categorie/extra/ → repairpods.nl/accessoires/

# Producten (1:1 mapping)
airpodshandel.nl/product/linker-gen-1-airpod/ → repairpods.nl/product/airpods-1e-generatie-links/
airpodshandel.nl/product/rechter-gen-1-airpod/ → repairpods.nl/product/airpods-1e-generatie-rechts/
airpodshandel.nl/product/case-gen-1-airpods/ → repairpods.nl/product/airpods-1e-generatie-oplaadcase/
# ... (alle producten)

# Catch-all voor verwijderde pagina's
airpodshandel.nl/* → repairpods.nl/losse-airpods/ (404 fallback)
```

#### Vermijden
- ❌ Redirect chains (A→B→C)
- ❌ Redirect loops
- ❌ Soft 404s
- ❌ Parameter URLs zonder redirect
